# Background presets

Drop JPEG files here to expose them as virtual backgrounds in the meeting UI.
The `BackgroundPicker` component references `/backgrounds/office.jpg` and
`/backgrounds/park.jpg`. Replace these with your own licensed images (or
point to different filenames in `src/components/BackgroundPicker.tsx`).

Recommended: 1920×1080 JPEG, under 500 KB.

This directory is served by Vite as static content at the `/backgrounds/` URL prefix.
